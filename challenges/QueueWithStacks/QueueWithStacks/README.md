# Pseudo Queue
#### *Author: Fix-It Felix*

------------------------------

## Problem Domain
Create a class called PsuedoQueue
Instantiate two empty stacks
Using these empty stacks and stack methods to emulate a queue
Create two methods:
	- Enqueue(value)
	- Dequeue()

------------------------------

## Inputs and Expected Outputs


**Enqueue**
| Input | Arg | Expected Output |
| :----------- | :----- | :----------- |
| [10]->[15]->[20] | [5] | [5]->[10]->[15]->[20] |

**Dequeue**
| Input | Arg | Expected Output |
| :----------- | :----- | :----------- |
| [10]->[15]->[20] | [20] | [10]->[15] |

------------------------------

## Big O


| Time | Space |
| :----------- | :----------- |
| O(1) | O(1) |
